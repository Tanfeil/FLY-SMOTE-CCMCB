code.shared.GAN

module documentation

Module for Conditional GAN (Generative Adversarial Network) implementation.

This module provides the implementation of a Conditional GAN where both the generator and discriminator are conditioned on labels to generate class-specific data. The main class is `ConditionalGAN`, which defines methods for building the generator and discriminator models, training the GAN, generating samples, and handling the weights of both models.
