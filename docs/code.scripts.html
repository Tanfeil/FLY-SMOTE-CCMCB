code.scripts

package documentation

Package for executing scripts related to the FLY-SMOTE-CCMCB project. This package includes various scripts for preprocessing, training, evaluating models, and visualizing results. It provides functionality for integrating Weights & Biases (W&B) for tracking experiments and generating plots for performance metrics. The scripts are organized by their respective purposes and are designed to be modular and reusable in different parts of the project workflow. Modules include: - Preprocessing of datasets - Training and evaluation of machine learning models - Visualization and plotting of results (accuracy, sensitivity, etc.) - Integration with W&B for experiment tracking

Module download_datasets Module for downloading and extracting datasets based on configuration.
Package plots Package for generating and saving plots for model evaluation in the FLY-SMOTE-CCMCB project.
Module runner_parallel This module provides functionality for running parameter variants in parallel across multiple tasks. It loads parameter combinations from a JSON file, executes the variants in separate processes, and logs the results.
