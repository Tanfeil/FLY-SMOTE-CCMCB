code.scripts.plots.wandb_plots_sensitivity

module documentation

Module for generating and saving sensitivity plots based on results from Weights & Biases (W&B) runs.

This module interacts with Weights & Biases to fetch the results of experiments, processes the data related to sensitivity tests, and generates line plots showing the sensitivity over different parameter values (e.g., k-value, r-value, g-value). The plots are saved in the specified output directory.
