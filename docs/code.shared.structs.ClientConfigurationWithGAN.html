class ClientConfigurationWithGAN(ClientConfiguration):

View In Hierarchy

Extension of ClientConfiguration for GAN-specific settings. Inherits from `ClientConfiguration` and adds additional attributes specific to training a GAN model, such as `g_value`, `global_gan_weights`, and `noise_dim`. Attributes: g_value (float): GAN-related parameter for model generation. global_gan_weights (list): Weights specific to GAN models that are shared globally. noise_dim (int): Dimension of the noise input for the GAN model.
