code.scripts.download_datasets

module documentation

Module for downloading and extracting datasets based on configuration.

This module handles the download of dataset files from specified URLs, extraction of ZIP files, and recursive extraction of nested ZIP files. It loads dataset configurations from a JSON file and manages the storage of datasets in a specified directory.
